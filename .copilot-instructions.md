# Copilot Instructions for Nodo Project

## Build Configuration
- This project uses **Conan presets**
- Always use: `cmake --preset=conan-debug` for configuration
- Always use: `cmake --build --preset=conan-debug` for building
- Never use `cmake --preset=default` - that's the wrong preset

## Build Task
When running the build task, use:
```bash
cmake --preset=conan-debug && cmake --build --preset=conan-debug
```

## Project Structure
- `nodo_core/` - Core library (no Qt dependencies)
- `nodo_studio/` - Qt6 GUI application
- `tests/` - Unit tests
