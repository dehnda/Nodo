Animations & Transitions

Subtle animations improve perceived performance and provide visual feedback. All animations use hardware-accelerated properties.

Standard Transitions 
/* Fast interactions (buttons, hover) */
transition: all 0.15s ease-out;

/* Standard UI elements */
transition: all 0.2s ease-out;

/* Slower, more deliberate (panels, modals) */
transition: all 0.3s ease-in-out;

/* Easing curves */
--ease-out: cubic-bezier(0.22, 1, 0.36, 1);
--ease-in-out: cubic-bezier(0.65, 0, 0.35, 1);
Context Menu Fade In 
@keyframes menuFadeIn {
  from {
    opacity: 0;
    transform: translateY(-8px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

animation: menuFadeIn 0.15s ease-out;
üé¨ Animation Principles
  • Use transform and opacity for hardware-accelerated animations
  • Keep animations under 300ms for snappy feel
  • Hover states: 150ms transition
  • Panel animations: 200-300ms with ease-in-out
  • Use spring-like easing for natural motion (ease-out)
  • Respect user's reduced motion preferences

Do's & Don'ts

Layout & Spacing

‚úì Do
  • Use consistent 8pt spacing system
  • Maintain visual hierarchy with size and weight
  • Group related elements with whitespace
  • Use 12px+ padding in clickable areas
‚úó Don't
  • Use arbitrary spacing values
  • Cram elements without breathing room
  • Mix spacing systems (stick to 4px multiples)
  • Create tap targets smaller than 40√ó40px

Color & Contrast

‚úì Do
  • Maintain 4.5:1 contrast ratio for text
  • Use accent color consistently for interactions
  • Layer with subtle opacity variations
  • Test colors in both light and dark modes
‚úó Don't
  • Use low-contrast text (#606068 on #1a1a1f)
  • Mix too many accent colors
  • Use category colors for general UI
  • Rely solely on color to convey meaning

Typography

‚úì Do
  • Use system font stack for performance
  • Limit to 2-3 font sizes per view
  • Use monospace for numeric values
  • Maintain 1.4+ line-height for readability
‚úó Don't
  • Load custom web fonts (performance)
  • Use all-caps for long text
  • Mix font weights randomly
  • Set line-height below 1.3
