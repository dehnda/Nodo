# Blast

**Category:** Utility

## Description

Delete geometry by group or selection

## Inputs

- **Input 0**: Geometry to process

## Parameters

### Group

**Group** (`string`)

Name of group to delete (leave empty for all)

- Default: `""`

**Delete** (`int`)

Type of elements to delete

- Default: `1` | Options: `Points`, `Primitives`

### Options

**Delete Non-Selected** (`int`)

Delete elements NOT in the group instead

- Default: `0`
