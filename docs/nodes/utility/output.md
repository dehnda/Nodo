# Output

**Category:** Utility

## Description

Mark geometry as final output

## Inputs

- **Input 0**: Geometry to process

## Parameters

### Output

**Output Name** (`string`)

Name identifier for this output

- Default: `"output1"`

**Render** (`int`)

Include this output in rendering

- Default: `1` | Options: `Off`, `On`

**Display** (`int`)

Show this output in the viewport

- Default: `1` | Options: `Off`, `On`
