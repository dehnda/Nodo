# Time

**Category:** Utility

## Description

Control time-dependent animations

## Inputs

This node generates geometry and requires no inputs.

## Parameters

### Time

**Frame** (`int`)

Current frame number for animation

- Default: `1` | Range: `1` to `240`

**FPS** (`float`)

Frames per second for time calculation

- Default: `24.0F` | Range: `1.0` to `120.0`

### Time Range

**Start Frame** (`int`)

First frame of animation range

- Default: `1` | Range: `1` to `1000`

**End Frame** (`int`)

Last frame of animation range

- Default: `240` | Range: `1` to `10000`
