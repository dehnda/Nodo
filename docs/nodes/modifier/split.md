# Split

**Category:** Modifier

## Description

Separate geometry by connectivity or attribute

## Inputs

- **Input 0**: Geometry to process

## Parameters

### Split

**Split By** (`int`)

- Default: `0` | Options: `Connectivity`, `Attribute`

**Attribute** (`string`)

- Default: `""`

### Output

**Create Groups** (`int`)

- Default: `1` | Options: `Off`, `On`

**Add Piece Attribute** (`int`)

Add integer 'piece' attribute to primitives

- Default: `1` | Options: `Off`, `On`

## Example Usage

Connect geometry to the input, then adjust parameters to modify the result.
